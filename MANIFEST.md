# cognis-devbox — what's preinstalled

**Languages:** Python 3, Node (nvm) + Bun + Deno, Go, Rust, Java (JDK), Ruby, PHP, .NET 8.
**Containers/Orch:** Docker, kubectl, Helm.
**IaC/Cloud:** Terraform, gcloud, AWS CLI, Azure CLI, GitHub CLI.
**AI/LLM:** Ollama, huggingface_hub, vLLM (where supported), llama.cpp build deps.
**Dev tooling:** git, jq, ripgrep, fzf, tmux, neovim, htop, build-essential.

Build targets: **Packer (QEMU/KVM qcow2)**, **Vagrant** (libvirt/VirtualBox), **cloud-init** (any cloud VM).
